

# 问题描述

<br/>
     一些正整数可以表示成一个或多个连续素数和的形式。那么一个正整数可以表示成多少种连续素数和的形式呢？例如： 53 有 2 种连续素数和的形式分别是 5+7+11+13+17 和 53. 正整数 41 有 3 种连续素数和的形式： 2+3+5+7+11+13,11+13+17 和 41. 整数 3 只有一种连续素数和的形式就是 3 。 20 就不能表示成连续素数的和。
