

# 题目描述

# 输入格式


<div class="ptx" lang="zh-CN">
输入文件包含不超过10000行，每行一个整数N。输入文件的最后一行是一个0.
</div>

# 输出格式


<div class="ptx" lang="zh-CN">
对每一个N（除了最后的0），输出一行即相应的最大值。
</div>

# 样例输入


<pre class="sio">5
10
0</pre>

# 样例输出


<pre class="sio">3
4</pre>
