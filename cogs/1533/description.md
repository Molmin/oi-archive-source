

# 题目描述



# 输入格式


<div class="content">
第一行为正整数n(n&lt;=32767)，表示该公司从成立一直到现在的天数，接下来的n行每行有一个正整数ai(ai&lt;=1000000)，表示第i天公司的营业额。
</div>

# 输出格式


<div class="content">
输出文件仅有一个正整数，即Sigma(每天最小的波动值) 。结果小于2^31 。
</div>

# 样例输入


<pre>6
  5
  1
  2
  5
  4
  6	</pre>

# 样例输出


<pre>12
  </pre>
