

# 问题描述


<p class="MsoNormal" style="text-align:left;" align="left">

</p>
<p class="MsoNormal" style="text-align:left;" align="left">
$$F_j=\sum_{i&lt;j}\frac{q_iq_j}{(i-j)^2}-\sum_{i&gt;j}\frac{q_iq_j}{(i-j)^2}$$
</p>
<p class="MsoNormal" style="text-align:left;" align="left">
令 $E_i=\frac{F_i}{q_i}$，求 $E_i$.
</p>
