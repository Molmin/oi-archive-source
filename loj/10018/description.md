
# 题目描述

将整数 $n$ 分成 $k$ 份，且每份不能为空，问有多少种不同的分法。当 $n=7, k=3$ 时，下面三种分法被认为是相同的：$1,1,5$; $1,5,1$; $5,1,1$

# 输入格式

一行两个数 $n$ , $k$。

# 输出格式

一行一个整数，即不同的分法数。

# 样例

#### 样例输入
```plain
7 3
```

#### 样例输出
```plain
4
```

#### 样例解释
四种分法为：$1,1,5$；$1,2,4$；$1,3,3$；$2,2,3$。

# 数据范围与提示

$6 \leq n \leq 200,$ $2 \leq k \leq 6$。
