
# 题目描述

给定 $ n $ 和 $ m $，求出所有顶点坐标满足 $ 0 \leq x < n, 0 \leq y < m $ 的格点三角形的面积和的两倍。答案模 $ 1004535809 $ 输出。

# 输入格式

一行两个整数 $ n $ 和 $ m $。

# 输出格式

一行一个整数表示答案。

# 样例

#### 样例输入 1
```plain
2 3
```

#### 样例输出 1
```plain
24
```

#### 样例输入 2
```plain
10 100
```

#### 样例输出 2
```plain
218427047
```

#### 样例输入 3
```plain
100 1000
```

#### 样例输出 3
```plain
938425419
```
