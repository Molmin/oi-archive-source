
# 题目描述

给定 $n,m$，求：

$$
\sum_{i_1=1}^{m}\sum_{i_2=1}^{m} \dots \sum_{i_n=1}^{m}\gcd(i_1,i_2,i_3, \dots i_n)
$$

答案对 $2^{64}$ 取模。


# 输入格式

一行两个整数 $n,m$。

# 输出格式

一行一个整数表示答案。

# 样例

#### 样例输入

```plain
10 10
```

#### 样例输出

```plain
10009889889
```

# 数据范围与提示

$1 \le n, m \le 10^{11}$
