
# 题目描述

给定两个正整数 $N,M$，你需要计算 $(\sum\limits_{i=1}^N\sum\limits_{j=1}^M \mu^2(\gcd(i,j))) \bmod 998244353$ 的结果。

# 输入格式

输入一行两个空格隔开的正整数，分别为 $N,M$。


# 输出格式

输出一行一个正整数表示求和取模后的结果。

# 样例

#### 样例输入1
```plain
2 3
```

#### 样例输出1
```plain
6
```

#### 样例输入2
```plain
57 71
```

#### 样例输出2
```plain
3763
```

# 数据范围与提示

$1 \leq N,M \leq 10^{13}$
