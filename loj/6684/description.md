
# 题目描述

定义一棵有根树为「奇树」，当且仅当其所有叶子深度都为奇数（根节点深度为 $1$）。

小 P 对奇树十分感兴趣，他想知道有多少棵 $n$ **个奇点**（不是 $n$ 个点）的有根无标号奇树（奇点深度为奇数。由上述定义根也是奇点）$\bmod 998244353$。

你需要对于每个 $n$ 输出答案。

# 输入格式

一行一个整数 $n$。

# 输出格式

输出 $n$ 行，第 $i$ 行表示有 $i$ 个奇点的答案。

# 样例

#### 样例输入
```plain
10
```
#### 样例输出
```plain
1
1
3
8
25
77
258
871
3049
10834
```

# 数据范围与提示

$n = 10^5$，对于 $50\%$ 的数据满足 $n = 40$。
