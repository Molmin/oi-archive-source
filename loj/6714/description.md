
# 题目描述

我们定义一个长度为 $m$ 的正整数序列 $\{ a_{i} \}(\forall i, a_{i} > 1)$ 的权值为 $\prod_{i = 1} ^ {m} a_{i}$；特殊地，空序列的权值为 $1$。

记权值为 $x$ 的序列个数为 $f(x)$。

给定正整数 $n$，你需要求出 $f(x)$ 的前缀和对 $998244353$ 取模的结果，即 $\sum_{i = 1} ^ {n} f(i) \bmod 998244353$。

# 输入格式

一个正整数 $n$。

# 输出格式

一个整数表示 $f(x)$ 的前缀和对 $998244353$ 取模的结果。

# 样例

#### 样例输入 1

```plain
10
```

#### 样例输出 1

```plain
19
```

#### 样例输入 2

```plain
233333
```

#### 样例输出 2

```plain
608089621
```
