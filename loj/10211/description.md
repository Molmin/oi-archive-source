
# 题目描述

**原题来自：Romania OI 2002**

求 $A^B$ 的所有约数之和 $\bmod\ 9901$。

# 输入格式

输入两个整数 $A,B$。

# 输出格式

输出答案 $\bmod 9901$。

# 样例

#### 样例输入
```plain
2 3
```
#### 样例输出
```plain
15
```
#### 样例说明
$2^3=8$，$8$ 的所有约数为 $1,2,4,8$，$1+2+4+8=15,15\bmod 9901=15$，因此输出 $15$。

# 数据范围与提示

对于全部数据，$0\le A,B\le 5\times 10^7$。
