
# 题目描述

**原题来自：[HackerRank Equations](https://www.hackerrank.com/challenges/EQUATIONS/problem)**

求不定方程：
$$\frac{1}{x}+\frac{1}{y}=\frac{1}{n!}$$
的正整数解 $(x,y)$ 的数目。

# 输入格式

一个整数 $n$。

# 输出格式

一个整数，表示有多少对 $(x,y)$ 满足题意。答案对 $10^9+7$ 取模。

# 样例

#### 样例输入
```plain
2
```
#### 样例输出
```plain
3
```
#### 样例说明
共有三个数对 $(x,y)$ 满足条件，分别是 $(3,6),(4,4)$ 和 $(6,3)$。

# 数据范围与提示

对于 $30\%$ 的数据，$n\le 100$；  
对于全部数据，$1\le n\le 10^6$。
