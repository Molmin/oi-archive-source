
# 题目描述

给定 $1\sim N$ 的一个全排列，试求它在所有 $1\sim N$ 全排列中的排名。结果对 $998244353$ 取模。

# 输入格式

第一行一个正整数 $N$。

第二行 $N$ 个正整数，表示 $1\sim N$ 的一种全排列。

# 输出格式

一行一个非负整数，表示答案对 $998244353$ 取模的值。

# 样例

#### 样例输入 1
```plain
3
2 1 3
```

#### 样例输出 1
```plain
3
```

#### 样例输入 2
```plain
4
1 2 4 3
```

#### 样例输出 2
```plain
2
```

# 数据范围与提示

对于 $5\%$ 数据，$1\le N\le 10$。  
对于 $20\%$ 数据，$1\le N\le 5000$。  
对于 $100\%$ 数据，$1\le N\le 10^6$。
