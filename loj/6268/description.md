
# 题目描述

令 $f(n)$ 表示将 $n$ 进行分拆的方案数。

例如，$f(4) = 1 + 1 + 1 + 1 = 1 + 1 + 2 = 1 + 3 = 2 + 2 = 4$ ，所以 $f(4) = 5$ 。

给 $n = 100000$ ，求 $f(1), f(2), ..., f(n)$ 对 $998244353$ 取模。

# 输入格式

第一行一个正整数 $n$ 。

# 输出格式

输出共 $n$ 行，第 $i$ 行输出 $f(i) \mod 998244353$ 。

# 样例

#### 样例输入

```plain
4
```

#### 样例输出

```plain
1
2
3
5
```
