# Input

<div class="content">共N+1行。
第一行有一个正整数N，表示区域的个数。
接下来有N-1行，每行两个整数u、v，表述区域u和区域v之间有一条道路。
最后一行有N个正整数，第i个正整数表示区域i的权值W(i)。
</div>

# Output

<div class="content">包含一个正整数，为最小的S(x, y)的值。
</div>

<div class="content"><span class="sampledata">5<br/>
