
# Description

<div class="content"><p><span style="font-size: medium">一个舞会有N个男孩（编号为1..N）和M个女孩（编号为1..M），一对男女能够组成一对跳舞当且仅当他们两个人互相认识。<br/>
我们把一种人定义成这个舞会的life：当且仅当如果他（她）不参加这个舞会，那么能够同时配对的最大舞伴对数会下降。<br/>
现在知道男生和女生之间的认识关系，需要你求出男生和女生中的是这个舞会的life的人的编号。</span></p>
<p></p></div>

# Input

<div class="content"><p><span style="font-size: medium">第一行3个整数N，M，K，表示N个男生，M个女生，K对关系。<br/>
接下来K行，每行两个整数a_i b_i，表示第a_i个男生和第b_i个女生相互认识。</span></p>
<p></p></div>

# Output

<div class="content"><p><span style="font-size: medium">首先输出所有男生中是这个舞会的life的男生的编号，一行一个，从小到大输出，然后输出女生的。</span></p>
<p></p></div>

# Sample Input

<div class="content"><span class="sampledata">4 4 4<br/>
2 1<br/>
3 2<br/>
4 3<br/>
4 4<br/>
<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">2<br/>
3<br/>
4<br/>
1<br/>
2<br/>
<br/>
<br/>
</span></div>

# Hint

<div class="content"><p></p><p><span style="font-size: medium">N,M&lt;=10^4，K&lt;=10^5<br/><br/>
</span></p><p></p></div>
