
# Description

<div>现在你可以用任意英文小写字母来代表它们。要求是使得给定的三个字符串中 </div>
<div>所有的&#34;?&#34;被你认定的字母代替后，各不相同且按字典序出现。问有多少种方式。 </div>
<div></div>
<p></p></div>
