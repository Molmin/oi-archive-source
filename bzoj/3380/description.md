
# Description

<div class="content"><div><span style="font-size: medium">    很少人知道其实奶牛非常喜欢到洞穴里面去探险。</span></div>
<div><span style="font-size: medium">    洞窟里有N(1≤N≤100)个洞室，由M(1≤M≤1000)条双向通道连接着它们．每对洞室间</span></div>
<div><span style="font-size: medium">至多只有一条双向通道．有K(1≤K≤14)个洞室，里面放有1捆干草．牛吃1捆干草，体重指数就会增加1．</span></div>
<div><span style="font-size: medium">    贪吃的贝茜要到洞窟里面探险．她希望能吃尽量多的干草，但每条通道有一个宽度阈值，如果体重指数超过相应的阈值，贝茜就会被卡祝她从洞窟1出发，体重指数为0．在洞里溜达一圈后，她要返回洞窟1．    那她最多能吃多少捆干草呢？注意，贝茜经过一个洞室，不一定非要吃掉里面的干草．</span></div></div>

# Input

<div class="content"><div><span style="font-size: medium">    第1行输入N，M，K,之后K行每行一个整数，表示在这个洞室放有一捆干草；接下来M行每行三个整数，表示一条双向通道的起点终点和宽度阈值．</span></div></div>

# Output

<div class="content"><div> </div>
<div><span style="font-size: medium">    最多能吃掉的干草数．</span></div></div>
