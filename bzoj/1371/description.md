
# Description

<div class="content">给出一个水池的底面积及高度。
现在把一些立方体放到其中，问最终水平面的高度.
注意:
1:水的密度为1.0
2:空气的影响忽略不计
3:立方体完全放到水池中。
4:立方体不会旋转且相互不会接触 。
