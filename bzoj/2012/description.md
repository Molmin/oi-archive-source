
# Description

<div class="content">给出N(2&lt;=N&lt;=50000)个长度为4的字符串，问有且仅有D(1&lt;=D&lt;=4)处不相同的字符串有几对。

</div>

# Input

<div class="content">
	第1行： N，D
	以下N行每行一个字符串

</div>

# Output

<div class="content">
	一个数：有多少对有且仅有处不相同的字符串。
</div>

# Sample Input

<div class="content"><span class="sampledata">4 2<br/>
0000<br/>
a010<br/>
0202<br/>
a0e2<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">3</span></div>
