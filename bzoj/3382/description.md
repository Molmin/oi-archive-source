
# Description

<div class="content"><div><span style="font-size: medium">    约翰的N(1≤N≤50000)只牛在一个黑魃魃的洞里探险．他们只能通过叫声交流．</span></div>
<div><span style="font-size: medium">    两只牛之间的曼哈顿距离决定了声音传播的时间．牛1与牛2交流，需要的时间为</span></div>
<div><span style="font-size: medium">|x1-x2|+|y1-y2|</span></div>
<div><span style="font-size: medium">其中x,y∈[_10^6，10^6]．</span></div>
<div><span style="font-size: medium">    那任意一对牛之间交流时间的最大值为多少？</span></div></div>

# Input

<div class="content"><div><span style="font-size: medium">    第1行输入N，接下来每行输入一只牛的坐标</span></div></div>

# Output

<div class="content"><div> </div>
<div><span style="font-size: medium">    交流时间最大值（即最大曼哈顿距离）．</span></div></div>

# Sample Input

<div class="content"><span class="sampledata">5<br/>
1 1<br/>
3 5<br/>
2 7<br/>
8 1<br/>
4 4</span></div>

# Sample Output

<div class="content"><span class="sampledata">12</span></div>
