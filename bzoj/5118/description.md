
# Description

<div class="content"><div>Fib定义为Fib(0)=0,Fib(1)=1,对于n≥2,Fib(n)=Fib(n-1)+Fib(n-2)</div>
<div>现给出N，求Fib(2^n).</div>
<p></p></div>

# Input

<div class="content"><div>本题有多组数据。第一行一个整数T，表示数据组数。</div>
<div>接下来T行每行一个整数N，含义如题目所示。</div>
<div>n≤10^15, T≤5</div>
<p></p></div>

# Output

<div class="content"><div>输出共T行，每行一个整数为所求答案。</div>
<div>由于答案可能过大，请将答案mod 1125899839733759后输出</div>
<p></p></div>

# Sample Input

<div class="content"><span class="sampledata">2<br/>
2<br/>
31</span></div>

# Sample Output

<div class="content"><span class="sampledata">3<br/>
343812777493853</span></div>
