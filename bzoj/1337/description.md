
# Description

<div class="content">给出平面上N个点,N&lt;=10^5.请求出一个半径最小的圆覆盖住所有的点
</div>

# Input

<div class="content">第一行给出数字N，现在N行，每行两个实数x,y表示其坐标.

</div>

# Output

<div class="content">输出最小半径，输出保留三位小数.
</div>

# Sample Input

<div class="content"><span class="sampledata">4<br/>
1 0<br/>
0 1<br/>
0 -1<br/>
-1 0<br/>
<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">1.000</span></div>
