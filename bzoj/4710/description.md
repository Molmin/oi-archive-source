
# Description

<div class="content"><div>JYY 带队参加了若干场ACM/ICPC 比赛，带回了许多土特产，要分给实验室的同学们。</div>
<div>JYY 想知道，把这些特产分给N 个同学，一共有多少种不同的分法？当然，JYY 不希望任</div>
<div>何一个同学因为没有拿到特产而感到失落，所以每个同学都必须至少分得一个特产。</div>
<div>例如，JYY 带来了2 袋麻花和1 袋包子，分给A 和B 两位同学，那么共有4 种不同的</div>
<div>分配方法：</div>
<div>A：麻花，B：麻花、包子</div>
<div>A：麻花、麻花，B：包子</div>
<div>A：包子，B：麻花、麻花</div>
<div>A：麻花、包子，B：麻花</div>
<div></div>
<p></p></div>

# Input

<div class="content"><div>输入数据第一行是同学的数量N 和特产的数量M。</div>
<div>第二行包含M 个整数，表示每一种特产的数量。</div>
<div>N, M 不超过1000，每一种特产的数量不超过1000</div>
<div></div>
<p></p></div>

# Output

<div class="content"><div>输出一行，不同分配方案的总数。由于输出结果可能非常巨大，你只需要输出最终结果</div>
<div>MOD 1,000,000,007 的数值就可以了。</div>
<div></div>
<p></p></div>

# Sample Input

<div class="content"><span class="sampledata">5 4<br/>
1 3 3 5</span></div>

# Sample Output

<div class="content"><span class="sampledata">384835</span></div>
