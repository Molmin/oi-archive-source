
# Description

<div class="content"><div>现在需要在一个N*M的网格图上种K棵树，树只能种在格子的四个角上，即实际上有(N+1)*(M+1)个位置可以种树。</div>
<div>这K棵树需要满足以下几个条件：</div>
<div>1、同一个位置最多只能种一棵树；</div>
<div>2、这K棵树需要在同一条直线上；</div>
<div>3、这K棵树两两距离都不能小于D。</div>
<div>现在请你计算满足条件的种树方案，由于答案可能很大，只需输出它对10^9取模的结果。</div>
<p></p></div>

# Input

<div class="content"><div>读入四个整数K，N，M，D。</div>
<p></p></div>

# Output

<div class="content"><div>输出仅一个整数，表示种树方案数对10^9取模的结果。</div>
<div>
<p></p>
</div></div>

# Sample Input

<div class="content"><span class="sampledata">2 4 4 1<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">300<br/>
</span></div>
