第二，如果不是质数，输出它最大的质因子是哪个。 <br/>

# Output

<div class="content"><p><font face="Times New Roman" size="4">第一行CAS(CAS&lt;=350，代表测试数据的组数) <br/>

对于每组测试数据：输出Prime，代表它是质数，或者输出它最大的质因子，代表它是和数 <br/>
</font></p></div>
