
# Description

<div class="content"><div>你要从(0,0)点走到(n,m)，每次只能往x轴或者y轴正方向移动一个单位距离。</div>
<div>从(i,j)移动到(i,j+1)的代价为ri,从(i,j)移动到(i+1,j)的代价为cj。</div>
<div>求最小代价。</div></div>

# Input

<div class="content"><div>第一行两个正整数n,m，第二行n+1正整数表示ri,第三行m+1个正整数表示ci。</div>
<div>n，m&lt;=10^5,0&lt;ri,ci&lt;=10^8。</div></div>

# Output

<div class="content"><div>一行一个整数，表示答案。</div></div>

# Sample Input

<div class="content"><span class="sampledata">2 3<br/>
5 3 7<br/>
7 2 5 6	<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">19</span></div>
