
# Description

<div class="content"><p>Byteotian Bit Bank (BBB) 拥有一套先进的货币系统,这个系统一共有n种面值的硬币,面值分别为b1, b2,..., bn. 但是每种硬币有数量限制,现在我们想要凑出面值k求最少要用多少个硬币.</p></div>

# Input

<div class="content"><p>第一行一个数 n, 1 &lt;= n &lt;= 200. 接下来一行 n 个整数b1, b2,..., bn, 1 &lt;= b1 &lt; b2 &lt; ... &lt; b n &lt;= 20 000, 第三行 n 个整数c1, c2,..., cn, 1 &lt;= ci &lt;= 20 000, 表示每种硬币的个数.最后一行一个数k – 表示要凑的面值数量, 1 &lt;= k &lt;= 20 000.</p></div>

# Output

<div class="content"><p>第一行一个数表示最少需要付的硬币数</p></div>

# Sample Input

<div class="content"><span class="sampledata">3<br/>
2 3 5<br/>
2 2 1<br/>
10<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">3</span></div>
