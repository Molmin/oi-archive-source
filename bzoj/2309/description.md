# Input

<div class="content"><p>第一行包含两个整数 n和q，表示字符串和附加任务的数量，中<br/>
间用一个空格隔开。 <br/>
接下来 n行，描述字符串，其中第 i 行包含一个字符串 Si。 <br/>
接下来 q行，描述附加任务，其中第 i 行包含两个整数Xi和Yi，中间用一个<br/>
空格隔开。</p></div>

<div class="content"><p>输出是第一行最大价值<br/>

</p></div>
