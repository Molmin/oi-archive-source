
# Description

<div class="content"><p>有n个球排成一列，每个球都有一个颜色，用A-Z的大写字母来表示，我们每次随机选出两个球ball1,ball2,使得后者染上前者的颜色，求期望操作多少次，才能使得所有球的颜色都一样？<br/>
</p></div>

# Input

<div class="content"><p>    一行一个字符串，表示球的颜色<br/>
   </p></div>

# Output

<div class="content"><p> 一行表示结果，精确到小数点后1位。<br/>
   </p></div>

</span></div>
