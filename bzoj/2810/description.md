# Input

<div class="content"><p>从标准输入读入数据。 <br/>
第一行包含两个被空格隔开的整数 W, H，表示广场的尺寸为W列H行。 <br/>
第二行包含一个整数N，表示忍者的数量。 <br/>
接下来N行中，第 i行有三个以空格分隔的整数 X<br/>
i,Yi, D i, 表示第i个忍者处在从左往右的 X<br/>
i列、从上往下的第Yi行，任何两个忍者不在同一个位置。第i个忍者面向的方向由Di</p>
<p>Di = 0，表示忍者向右； <br/>
Di = 1，表示忍者向上； <br/>
Di = 2，表示忍者向左； <br/>
Di = 3，表示忍者向下。</p>
