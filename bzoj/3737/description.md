
# Description

<div class="content"><p>给定n求所有满足phi(x)=n的x</p></div>

# Input

<div class="content"><p>第一行一个数T(T&lt;=5)<br/>
接下来T行,每行一个数n(n&lt;=10^10)</p></div>

每组一个数m表示满足的个数<br/>
接下来一行升序输出m个数xi,若m为0输出一个空行</p></div>
