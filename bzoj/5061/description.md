# Input

<div class="content"><div>第一行一个数t表示数据组数。</div>
<div>接下来t行每行三个数m,n,x,其中x为实数，小数点后位数不超过3，且0&lt;=x&lt;=6.283。</div>
<div>t&lt;=1000,m&lt;=30,n&lt;=10^9</div>
<div></div>
<p></p></div>

# Output

<div class="content"><div>输出t行表示答案。你的答案与标准答案绝对或相对误差不超过0.01时视为正确。</div>
<div></div>
<p></p></div>
