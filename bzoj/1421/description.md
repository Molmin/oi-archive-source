
# Description

<div class="content"><p>求3&lt;=(3*x) Mod 5&lt;=4的最小正整数解是多少？3！这个很简单. 但当把数字放得很大呢?</p></div>

# Input

<div class="content"><div>
<div>第一行一个整数。T代表数据组数。</div>
<div>接下来T行，每行4个数，M，D，L，R表示L&lt;=(D*x Mod M)&lt;=R。</div>
<div>T&lt;=100,1&lt;=M、D、L、R&lt;=1000000000</div>
</div></div>

# Output

<div class="content"><p>共T行，表示x的最小正整数解，若无解则输出-1。</p></div>

# Sample Input

<div class="content"><span class="sampledata">1<br/>
5 3 3 4</span></div>

# Sample Output

<div class="content"><span class="sampledata">1</span></div>
