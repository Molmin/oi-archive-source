
# Description

</div>

# Input

<div class="content">	第一行： N，D
	第二行： N个数，表示每块砖的长度。
</div>

# Output

<div class="content">	方案数。输出要mod1000000009</div>

# Sample Input

<div class="content"><span class="sampledata">4 1<br/>
1 2 3 100<br/>
</span></div>
