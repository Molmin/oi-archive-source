
# Description

<div>这一操作，直到A非降为止。求有多少种不同的操作方案，答案模10^9+7。</div>
<div></div>
<p></p></div>

<div class="content"><div>第一行一个整数n。</div>
<div>接下来一行n个整数，描述A。</div>
<p></p></div>

# Output

<div class="content"><p>一行一个整数，描述答案。</p>
<p></p></div>
