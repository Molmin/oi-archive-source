色线条的总长度即为所求. <img border="0" alt="" src="/source/bzoj/1043/img/aHR0cHM6Ly9seWRzeS5jb20vSnVkZ2VPbmxpbmUvaW1hZ2VzLzEwNDMuanBn.jpg"/> </span></p></div>

# Input

<div class="content"><p>　　第一行为1个整数n,N&lt;=1000<br/>

<br/>
</p></div>
