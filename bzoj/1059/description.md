
# Description

<div class="content"><div>　　小Q是一个非常聪明的孩子，除了国际象棋，他还很喜欢玩一个电脑益智游戏——矩阵游戏。矩阵游戏在一个N</div>
<div>*N黑白方阵进行（如同国际象棋一般，只是颜色是随意的）。每次可以对该矩阵进行两种操作：行交换操作：选择</div>
<div>矩阵的任意两行，交换这两行（即交换对应格子的颜色）列交换操作：选择矩阵的任意行列，交换这两列（即交换</div>
<div>对应格子的颜色）游戏的目标，即通过若干次操作，使得方阵的主对角线(左上角到右下角的连线)上的格子均为黑</div>
<div>色。对于某些关卡，小Q百思不得其解，以致他开始怀疑这些关卡是不是根本就是无解的！！于是小Q决定写一个程</div>
<div>序来判断这些关卡是否有解。</div></div>

# Input

<div class="content"><div>　　第一行包含一个整数T，表示数据的组数。接下来包含T组数据，每组数据第一行为一个整数N，表示方阵的大</div>
<div>小；接下来N行为一个N*N的01矩阵（0表示白色，1表示黑色）。</div></div>

# Output

<div class="content"><p>　　输出文件应包含T行。对于每一组数据，如果该关卡有解，输出一行Yes；否则输出一行No。</p></div>

# Sample Input

<div class="content"><span class="sampledata">2<br/>
2<br/>
0 0<br/>
0 1<br/>
3<br/>
0 0 1<br/>
0 1 0<br/>
1 0 0</span></div>

# Sample Output

<div class="content"><span class="sampledata">No<br/>
Yes<br/>
【数据规模】<br/>
对于100%的数据，N ≤ 200</span></div>
