
# Description

<div class="content">平面上最开始只包含3个点，然后还会依次出现N个点。每新增一个点，请你求出包含这些点的周长最小的多边形的面积（也就是凸包的面积）。 

</div>

# Input

<div class="content">第一行为6个整数，表示最初的三个点的坐标(x1,y1),(x2,y2),(x3,y3)。 
第二行仅一个数N。 
以下N行，每行两个整数(xi,yi)，表示新增的点的坐标。 

</div>

# Output

<div class="content">对于每次新增点的操作，请你输出新增完此点后当前的凸包的面积的两倍。 

</div>

# Sample Input

<div class="content"><span class="sampledata">0 0 0 2 2 0<br/>
3<br/>
2 2<br/>
1 1<br/>
2 4<br/>
<br/>
</span></div>

# Sample Output

<div class="content"><span class="sampledata">8<br/>
8<br/>
12<br/>
<br/>
</span></div>

# Hint

<div class="content"><p>N&lt;=100000<br/>
点的坐标巨大,目测在[-10^9,10^9]</p></div>
