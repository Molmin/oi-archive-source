
# Content

给出n个矩形的左下角和右上角的坐标，求矩形面积的并

# Standard Input

多组输入(小于等于300组)，每组首先一个n(n<=10000)，代表矩阵的数量

其次是n行，每行四个数字，代表每个矩阵左上角(x1,y1)坐标和右下角(x2,y2)坐标

n=0时读入结束。

# Standard Output

每组一行一个数字，矩形面积的并
